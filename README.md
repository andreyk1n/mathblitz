# MathBlitz — Тренажер арифметики

Мінімалістичний браузерний тренажер для практики арифметики на час. Без залежностей, без бекенду — один HTML-файл.

---

## Можливості

**Операції**
- Додавання, віднімання, множення, ділення
- Режим «Мікс» — усі операції в одній сесії

**Розрядність чисел**
- 1, 2 або 3 цифри у кожному числі
- Хард-мікс — розрядність змінюється щоразу випадково

**Режими гри**
- На час: 1, 3 або 5 хвилин
- До першої помилки — гра зупиняється одразу після неправильної відповіді

**Тип відповіді**
- Ввести число з клавіатури (підтверджувати Enter або кнопкою OK)
- Вибрати з трьох варіантів

**Комбо-система**
- Серія правильних відповідей множить нагороду: ×1, ×2, ×3...
- Помилка скидає комбо до ×1

**Таймер**
- Кругова дуга з анімацією
- Змінює колір: білий → жовтий → червоний
- Останні 10 секунд — звуковий біп-сигнал

**Звук та тема**
- Звукові ефекти для правильних і неправильних відповідей
- Біп-сигнал таймера можна вимкнути в налаштуваннях
- Повна підтримка світлої та темної теми (зберігається між сесіями)

**Результати**
- Очки, кількість правильних/неправильних відповідей, точність у відсотках
- Ранг за результатом: 🏆 Блискуче / 🥇 Чудово / 🥈 Непогано / 🥉 Продовжуй

---

## Технології

- Чистий HTML5, CSS3, JavaScript (ES6+) — без фреймворків
- CSS за методологією BEM
- Web Audio API для звукових ефектів
- `localStorage` для збереження теми
- SEO: мета-теги, Open Graph, canonical URL
